# Решения

## Почему Web App, а не просто ссылка?
- Открывается ВНУТРИ Telegram — wow-эффект
- Полноэкранные анимации
- Чувствуется как подарок, а не как спам

## Почему Vercel для webapp?
- Бесплатно
- Мгновенный деплой из Git
- HTTPS из коробки (обязателен для Telegram Web App)

## Почему python-telegram-bot?
- Простой и понятный
- Async из коробки
- Хорошая документация

## Почему бот, а не просто Web App?
- Бот останется навсегда — можно добавлять фичи
- Личный канал общения
- Можно присылать напоминания, сюрпризы
