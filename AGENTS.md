# AGENTS.md — Valentine Bot

## Проект
Телеграм-бот с Web App валентинкой для моей девушки.
Бот отправляет кнопку → открывается валентинка прямо в Telegram → внутри ссылка на сайт.

## Стек
- **Bot**: Python, python-telegram-bot v20+
- **Web App**: Vanilla HTML/CSS/JS + Telegram Web App SDK
- **Хостинг webapp**: Vercel (бесплатно)
- **Хостинг бота**: локально или Railway/Render

## Структура
- `bot/` — Telegram бот на Python
- `webapp/` — HTML валентинка (деплоится на Vercel)
- `docs/` — документация и контекст

## Команды бота
- `/start` — отправляет кнопку "Открой валентинку" (WebApp)

## Переменные окружения
- `BOT_TOKEN` — токен из BotFather
- `WEBAPP_URL` — URL задеплоенной валентинки на Vercel

## Будущее использование
Бот задуман как долгосрочный — можно добавлять новые фичи:
- Напоминания о датах
- Фотоальбом
- Мини-игры для двоих
